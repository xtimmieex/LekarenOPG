# Lekáreň OPG

## Timea – Lekáreň
- Možnosť vyradiť lieky po expirácii
- **Analytika:**
    - Vypísať všetky lieky
    - Vypísať lieky po expirácii

## Eli – Liek
Každý liek má tieto vlastnosti:
- **Názov**
- **Účinná látka**
- **Dátum expirácie**
- **Cena**

## Dano – Sklad liekov
- **Ovládanie:**
    - Naskladniť liek
    - Predať liek pacientovi
    - Možnosť zoradiť výpis podľa kategórií (**účinná látka**)
